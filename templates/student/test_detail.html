{% else %} {# Подробные описания (6–7 предложений) для каждого теста #} {% if test.slug == 'mbti' %}

Короткая версия MBTI оценивает ваши предпочтения в четырёх дихотомиях: экстраверсия/интроверсия, ощущение/интуиция, мышление/чувство и суждение/восприятие. На основе ответов формируется четырёхбуквенный тип (например, INTJ), описывающий привычные способы получать информацию, принимать решения и взаимодействовать с миром. Тест не ставит «ярлыков» и не является клинической диагностикой — он помогает осознать свои естественные предпочтения. Профиль можно использовать для выбора ролей в проекте, способов учёбы и коммуникации. Узнавая свой тип и типы других людей, легче предотвращать конфликты и договариваться. В результате вы получите сильные стороны, зоны роста и советы по эффективной подготовке к экзаменам.

{% elif test.slug == 'klimov' %}

Дифференциально-диагностический опросник Е. А. Климова показывает склонность к типам труда: «человек-человек», «человек-техника», «человек-природа», «человек-знаковая система», «человек-художественный образ». Методика помогает понять, с какими объектами вам комфортнее работать и в какой логике вы лучше раскрываетесь. Полезно при выборе профиля обучения, элективов и внеурочной активности. В отчёте — ведущий тип и близкие альтернативы, а также примеры факультетов и стартовых ролей, где эти склонности востребованы.

{% elif test.slug == 'kos2' %}

КОС-2 измеряет коммуникативные и организаторские склонности — качества, важные для командной работы. Коммуникативная часть показывает, насколько легко вы устанавливаете контакты и убеждаете; организаторская — готовность брать ответственность и координировать процесс. Понимание профиля помогает выбрать роль в проектах (инициатор, координатор, исполнитель). В результате — уровни по обеим шкалам и рекомендации для развития.

{% elif test.slug == 'interests' %}

«Карта интересов» выявляет сферы деятельности, которые вам действительно нравятся, и помогает сузить поиск. Вопросы отделяют устойчивые предпочтения от ситуативных симпатий. По итогам — топ-сферы и связанные с ними примеры профессий/курсов. Полезно при выборе элективов, олимпиад и кружков; подскажем, какие предметы усилят сильные стороны. Это первый шаг к осознанному выбору вуза и специальности.

{% elif test.slug == 'thinking' %}

Тест на тип мышления показывает, какие способы решения задач преобладают: наглядно-образные, абстрактно-логические, практико-ориентированные и др. Понимание стиля помогает подобрать методы подготовки: схемы и карточки, подробные конспекты, задачи-кейсы или визуализация. В отчёте — сильные стороны, возможные трудности и конкретные советы для учёбы и проектов.

{% elif test.slug == 'child_type' %}

Методика оценивает особенности личности школьника: темперамент, стиль взаимодействия и самостоятельность. Помогает понять, какие условия обучения и поддержки дадут наилучший эффект. В результате — сильные стороны и рекомендации по мотивации, коммуникации и распределению нагрузки. Методика не ставит диагнозов и предназначена для образовательной навигации; при необходимости её дополняют консультацией психолога.

{% elif test.slug == 'bennett' %}

Тест Беннета оценивает пространственное воображение, внимание к деталям и техническую сообразительность — ключевые качества для инженерных, архитектурных и дизайнерских направлений. По результатам вы узнаете свой уровень и получите рекомендации по развитию пространственного мышления и выбору подходящих учебных проектов.

{% elif test.slug == 'cdi' %}

CDI — скрининговый опросник эмоционального состояния для детей 7–17 лет. Он помогает заметить признаки возможных трудностей, но не является диагнозом. Прохождение рекомендуется с последующей консультацией школьного психолога. Результаты видны только специалисту и используются для планирования поддержки; в кабинете ученика фиксируется лишь факт прохождения и дата.

{% else %}

Этот тест помогает понять ваши интересы, склонности и сильные стороны и подобрать способы обучения. Итоговый профиль пригодится при выборе элективов, кружков и направления поступления. Вы получите интерпретацию результатов и практические рекомендации — надёжную основу для осознанного образовательного маршрута.
